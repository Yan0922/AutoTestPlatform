## 二、功能模块

### 2.1 用户与权限管理
- 注册 / 登录 / 登出（JWT）
- 角色：管理员 / 普通用户
- 个人信息修改、密码重置

### 2.2 模型管理
| 字段 | 说明 |
|------|------|
| 模型名称/版本 | 标识模型 |
| API 地址 | HTTP 接口 URL |
| 请求格式配置 | headers、参数映射 |
| 响应字段映射 | 识别文本在响应 JSON 中的路径 |
| 状态 | 启用 / 禁用 |

- 连通性测试（ping）
- 增删改查

### 2.3 数据集管理
- 数据集基本信息（名称、语言、领域）
- **批量导入**：上传 zip 包（内含 wav + txt 对）
- 单条上传：单个 wav + 参考文本
- 条目列表：在线播放音频、编辑参考文本
- 导出 zip

### 2.4 测试任务
- 创建任务：选模型 + 选数据集 + 配置并发/超时
- 状态流转：`待执行 → 执行中 → 已完成 / 失败`
- 支持暂停 / 取消
- 实时进度显示
- Celery 异步执行，逐条调用 ASR API

### 2.5 测试结果与评测
**评测指标：**
- **WER**（词错误率）
- **CER**（字符错误率）
- **RTF**（实时率 = 推理耗时 / 音频时长）
- 平均推理耗时、成功率

**结果明细：**
- 参考文本 vs 识别文本（差异高亮）
- 音频在线播放
- 单条 WER / CER / RTF

**导出：** Excel（明细+汇总）、JSON

### 2.6 对比分析
- 多模型横向对比（同数据集）：表格 + 雷达图 + 柱状图
- 历史趋势：同模型跨时间 WER/CER 折线图

### 2.7 数据统计看板（Dashboard）
- 任务总数、模型数、数据集数
- 平均 WER/CER 趋势图
- 最近任务列表
- 各模型性能排行

## 六、部署方案（Windows 单机）

### 环境依赖
- Python 3.10+
- Node.js 18+
- Redis for Windows（[下载地址](https://github.com/tporadowski/redis/releases)）

### 安装步骤
```bat
# 1. 安装后端依赖
pip install -r backend/requirements.txt

# 2. 初始化数据库
python backend/manage.py migrate
python backend/manage.py createsuperuser

# 3. 安装前端依赖并构建
cd frontend && npm install && npm run build

# 4. 收集静态文件
python backend/manage.py collectstatic
```

### 启动（start.bat 一键完成）
```
Redis      → redis-server.exe
Celery     → celery -A core worker --pool=solo -l info
Django     → waitress-serve --host=0.0.0.0 --port=8000 core.wsgi:application
访问地址   → http://localhost:8000
```
